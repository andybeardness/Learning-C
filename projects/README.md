# Учебные творения

Тут собираю мини-проекты, которые приходили в голову во время чтения книги

----

### Звёздочки?

Это типа сложность

- ⭐ -- легчайше
- ⭐⭐ -- легко
- ⭐⭐⭐ -- средне
- ⭐⭐⭐⭐ -- трудоёмко
- ⭐⭐⭐⭐⭐ -- попотел

----
